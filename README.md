# Quiz Trainer

## Студент: Зинченко Мария Сергеевна, Группа: Б9123-09.03.03ПИКД(6).

## Тема викторины: Основы Jetpack Compose

### Описание:
Приложение-викторина на Jetpack Compose с вопросами по основам Kotlin и Android разработки.

### Функционал:
- Экран приветствия с описанием
- 4 вопроса с выбором ответа  
- Подсчёт правильных ответов  
- Экран результата с процентами  
- Возможность пройти заново  
- Соответствие всем требованиям задания
